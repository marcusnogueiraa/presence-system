Sobre as Turmas

Cada turma criada receberá um ID único (UUID) que será usado para registrar as presenças via ESP32.
O que você pode fazer com uma turma:
  • Adicionar alunos com seus respectivos cartões RFID
  • Visualizar o histórico completo de presenças
  • Registrar presenças em tempo real via API
  • Gerar relatórios de frequência dos alunos
Após criar a turma, anote o ID único que será gerado, pois ele será necessário para integração com o ESP32.
